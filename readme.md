# GRACkLe Guidelines

[![CC BY-NC-SA 4.0][cc-by-nc-sa-shield]][cc-by-nc-sa]

Guidelines for the Geochemistry and Radioisotope Analytical and Computational Laboratory ([GRACkLe](https://grackletu.github.io)) at Trinity University.
The document summarizes the overarching goals and philosophy of the lab, recommendations on how to get involved, my expectations of lab members, and some reasonable expectations lab members should have of me.
